# Vikingo y el Cruce del Río

**Descripción**: Este es un juego de lógica desarrollado en Java, donde debes ayudar al vikingo a cruzar el río junto con Caperucita, un lobo y unas uvas, evitando que el lobo se coma a Caperucita o que Caperucita se coma las uvas.

## Características
- **Desarrollado en**: Java
- **IDE recomendado**: IntelliJ

## Requisitos
- Java JDK 11
- IntelliJ IDEA

## Instalación
1. Clona el repositorio en tu sistema.
2. Abre el proyecto con IntelliJ.
3. Ejecuta la clase principal para iniciar el juego.

## Instrucciones del Juego
- El objetivo es cruzar todos los personajes al otro lado del río sin incidentes.
